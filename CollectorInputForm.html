How To Use

  1. Define a set of territories below.
  2. For each territory in the set a new single-territory continent is created with the 'Individual Bonus'.
  3. For each pair of territory in the set a new single-territory continent is created with the 'Individual Bonus'.
A set of N territories has N*(N-1)/2 pairs.

Leave values of '0' for no bonus of that type.
